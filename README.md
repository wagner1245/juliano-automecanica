# Juliano Automecânica Primeria versã

Sistema de gestão para oficina mecânica desenvolvido em **Python**, com interface gráfica em **Tkinter** e banco de dados **SQLite**.

## Sobre o projeto

O sistema foi criado para auxiliar no gerenciamento de uma oficina mecânica, permitindo o cadastro de clientes, serviços, ordens de serviço e controle básico financeiro.

Este projeto está em desenvolvimento e recebe melhorias contínuas.

## Funcionalidades atuais

- Cadastro de clientes
- Busca de clientes por **CPF**, **placa** ou **nome**
- Validação de CPF e placa duplicados
- Cadastro e edição de serviços
- Cálculo automático de valores
- Tela de ordens de serviço
- Resumo financeiro simples
- Interface gráfica desktop com Tkinter
- Banco de dados local com SQLite

## Tecnologias utilizadas

- Python
- Tkinter
- SQLite
- Pillow
